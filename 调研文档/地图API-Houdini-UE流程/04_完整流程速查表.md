# 04 完整流程速查表

## 最规范的完整流程（2024~2025 现状）

```
┌─────────────────────────────────────────────────────────────┐
│  STEP 1：框选数据 · 数据获取                                   │
│                                                             │
│  overpass-turbo.eu  →  框选地图范围  →  导出 .osm 文件        │
│  USGS EarthExplorer →  下载 DEM 高程  →  GeoTIFF             │
└────────────────────────┬────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────────┐
│  STEP 2：Houdini 处理                                        │
│                                                             │
│  Labs OSM Import SOP                                        │
│    ├── 建筑轮廓 → PolyExtrude → Labs OSM Buildings           │
│    ├── 道路线段 → Labs OSM Roads → 路面多边形                 │
│    └── 地块分区 → Landuse 贴地面                              │
│                                                             │
│  Height Field File SOP（导入 DEM）                           │
│    ├── Height Field Resize → 统一分辨率                      │
│    ├── Height Field Project → 道路压入地形                   │
│    └── Height Field Layer → 地表材质分层                     │
│                                                             │
│  打包为 HDA（暴露 bbox / 高度系数 / 密度 等参数）               │
└────────────────────────┬────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────────┐
│  STEP 3：UE5 整合                                            │
│                                                             │
│  Houdini Engine Plugin 加载 .hda                            │
│    ├── 实时调参 Cook → 场景更新                               │
│    └── Bake → Static Mesh / Landscape                       │
│                                                             │
│  可选叠加：                                                  │
│    ├── Cesium for Unreal → 流式真实底图                      │
│    ├── PCG Graph → 运行时程序化植被/道具                      │
│    └── World Partition → 无缝大世界分区                      │
└─────────────────────────────────────────────────────────────┘
```

## 常见坑与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| OSM 建筑无高度 | 该地区 OSM 数据缺少 `building:levels` 属性 | 在 Houdini 中用随机/规则系统补全默认高度 |
| 地形与建筑错位 | 坐标投影不一致 | 统一使用 UTM 坐标系，或在 Houdini 中手动对齐参考点 |
| HDA 导入 UE 后 Scale 100倍 | Houdini 单位 vs UE cm 问题 | 在 HDA Output 节点加 `transform scale 0.01` 或 UE 导入时 Scale=100 |
| Mapbox 节点失效 | SideFX 已于2024年10月移除 | 改用 OSM 或自行下载高程数据 |
| Cesium 无法离线使用 | 流式服务，无本地缓存 | 仅用于在线展示，离线场景用静态导出方案 |
| HDA Cook 速度慢 | 几何体量过大 | 在 Houdini 中做好 LOD 分级，UE 内只 Cook 当前 LOD |

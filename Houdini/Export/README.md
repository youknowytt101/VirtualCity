# Export

> 存放 Houdini 静态导出结果，如 FBX、OBJ、Alembic、Heightmap、参数 JSON。

---

## 推荐命名

```text
VC_{area_id}_buildings_v001.fbx
VC_{area_id}_roads_v001.fbx
VC_{area_id}_landscape_height_v001.png
VC_{area_id}_params_v001.json
```

---

## 用途

即使主线使用 HDA，也建议保留静态导出备份，方便：

- 快速导入 UE5 测试。
- 排查 Houdini Engine 问题。
- 做版本对比。
